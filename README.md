# 🎵 Six Music

Six Music é um bot de música para Discord desenvolvido para oferecer reprodução estável sem depender do YouTube como fonte de áudio.

## ✨ Recursos

- Spotify (músicas, álbuns e playlists)
- SoundCloud
- Bandcamp
- Audius
- Rádios HTTP
- Painel interativo
- Fila persistente
- Estatísticas
- Backup automático
- Português e Inglês
- Sistema de segurança por canal de voz
