Commands Documentation

Complete list of Six Music commands and their functionality.

🎵 Playback

/play - Play a song
/skip - Skip current song
/stop - Stop playback
/pause - Pause song
/resume - Resume song
/nowplaying - Show current track

📂 Queue

/queue - Show queue
/clear - Clear queue
/shuffle - Shuffle queue
/history - Show recently played songs

💜 Favorites

/favorite - Save current song
/favorites - Show favorites
/playfavorite - Play a saved favorite
/removefavorite - Remove a favorite

📁 Playlists

/saveplaylist - Save queue as playlist
/loadplaylist - Load saved playlist
/playlists - Show saved playlists
/removeplaylist - Delete a playlist

⚡ Smart Systems

/autoplay - Enable autoplay
/voteskip - Vote to skip songs
/loop - Toggle loop mode

📊 Utility

/stats - Show bot statistics
/invite - Get bot invite link
/support - Get support server link
/ping - Check bot latency
